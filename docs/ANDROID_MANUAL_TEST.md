# Android 手工测试清单

本文用于测试 Focus Timer 的 Android APK。目标是确认最新构建能在真实设备上完成核心流程，并把问题记录到可复现的程度。

## 测试对象

- APK 路径：`build\app\outputs\flutter-apk\app-release.apk`
- 当前已知大小：约 59.6 MB
- 当前已知 SHA256：`5347E46C8E7C660471B9507759892C2E08E5D7AA5E6BF59120A433401EF04216`
- 构建命令：`.\tools\build_apk.ps1 -Mode release`
- 注意：当前 release APK 仍使用 debug keystore，适合本地测试，不适合作为正式发布包。

## 安装与启动

### 1. 全新安装

步骤：

1. 卸载旧版本 Focus Timer。
2. 安装最新 `app-release.apk`。
3. 首次打开应用。

预期：

- 应用可以正常启动。
- 不出现崩溃、白屏、长时间黑屏。
- 不主动请求无关权限，例如定位、联系人、短信、通话、账号。
- 如系统要求通知、相机、精确提醒等权限，应能看到合理提示。

### 2. 覆盖安装

步骤：

1. 保留旧版本应用和数据。
2. 直接安装最新 APK。
3. 打开应用，查看旧数据是否还在。

预期：

- 应用启动正常。
- 旧的分组、待办、计时记录、打卡记录能正常读取。
- 数据库迁移不报错。
