# 手机用户测试指南

这份文档给拿手机测试的人使用。测试时不用懂代码，也不用连接电脑，只需要按步骤操作，并记录结果。

测试方式：

- 每个用例按顺序操作。
- 如果看到的结果和“期望结果”一致，就标记通过。
- 如果不一致，记录手机型号、Android 版本、截图或录屏、复现步骤。


建议先准备：

- 手机里有一个短视频文件，用于测试“选择已有视频”。
- 手机允许安装本地 APK。
- 如果系统弹出通知、相机、提醒权限，请按测试用例里的说明处理。

### 1.1 全新安装

操作步骤：

1. 如果手机里已经安装 Focus Timer，先卸载。
