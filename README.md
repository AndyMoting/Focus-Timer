# Focus Timer

开源生产力应用 — 计时、待办、分析，全在本地。

- ⏱️ 计时器（番茄钟、自由计时、倒计时等）
- 📝 待办列表（多分组、优先级、回收站）
- 🔥 热力图分析
- 📹 视频学习打卡
- 💾 本地 SQLite，无云同步

## 快速开始

```bash
git clone https://github.com/AndyMoting/Focus-Timer.git
cd Focus-Timer
flutter pub get
flutter build apk --release
```

Android 11+
